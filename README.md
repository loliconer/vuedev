## vuedev

> vuedev is a tooling for developing various types of Vue.js project.

## Supported project types

- DApp
- Electron
- PWA
- SPA
- MPA
- Admin System
- Responsive
- International
- Server
- Test
- Component
- VuePress

## Getting Started

Install:

```bash
npm install -g vuedev
```

Create a project:

```bash
vuedev crate my-project
```
